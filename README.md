# 🌐 Meninas na Ciência IFB

Site desenvolvido para o projeto **Meninas na Ciência IFB**, com o objetivo de divulgar iniciativas, eventos e incentivar a participação feminina nas áreas de tecnologia e ciência.

---

## 📌 Sobre o projeto

O site Meninas na Ciência IFB foi desenvolvido para divulgar e fortalecer as ações do projeto, que tem como objetivo incentivar a participação feminina nas áreas de ciência e tecnologia.

A plataforma reúne informações sobre o projeto, suas iniciativas, eventos e formas de participação, funcionando como um canal de comunicação e visibilidade para a comunidade.

Além disso, o site contribui para ampliar o alcance das atividades do projeto, promovendo o acesso à informação e estimulando o interesse de meninas pela área de tecnologia.

## 💻 Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/Emilyms-tech/Site-do-projeto-meninas-na-Ciencia-IFB.git
```

2. Acesse a pasta do projeto:

```
cd Site-do-projeto-meninas-na-Ciencia-IFB
```

3. Abra o arquivo `index.html` no navegador.
