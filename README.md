# site-do-projeto Meninas na Ciencia-IFB
<p>O projeto de extensão Meninas na Ciência possui a intenção de incentivar a participação feminina nas áreas de STEAM (Ciência, Tecnologia, Engenharia, Arte e Matemática)e a landing page faz a apresentação das orientadoras, das alunas, os objetivos do projeto e trabalhos publicados. </p>
